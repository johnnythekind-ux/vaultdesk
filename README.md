# VaultDesk

VaultDesk is a full-stack workspace management application that allows users to securely create, organize, and manage personal records using an authenticated, database-driven system.

## 🚀 Overview

VaultDesk demonstrates a complete implementation of:

- User authentication
- Secure, user-scoped data access
- Full CRUD operations (Create, Read, Update, Delete)
- Server-side data handling with Next.js
- Supabase database integration with Row Level Security (RLS)

Each user can create and manage their own workspaces, ensuring all data is isolated and protected.

## ⚙️ Local Development

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/vaultdesk.git
cd vaultdesk
```

Install dependencies:

```bash
npm install
```

Create a `.env.local` file:

```env
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_PUBLISHABLE_KEY=your_key
SUPABASE_SERVICE_ROLE_KEY=your_service_role_key
```

Run the development server:

```bash
npm run dev
```

## 🔐 Database Design

Core table:

### workspaces

- id
- user_id
- name
- description
- created_at
- updated_at

All queries are scoped using:

```ts
.eq("user_id", user.id)
```

This ensures users can only access their own data.
